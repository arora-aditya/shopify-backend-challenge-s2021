## Pre-requisites to run locally
1. NodeJS (v12.20.0 or above)
2. Yarn (v1.22.4 or above)
3. EveryPixel API key

## Steps to run
1. `git clone <repo>`
2. Save secrets in `.env` file
3. `yarn install`
4. `yarn dev`
